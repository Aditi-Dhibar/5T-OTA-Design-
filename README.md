This repository contains **Assignment 3**, which focuses on the design and analysis of a **5-Transistor Operational Transconductance Amplifier (5T OTA)** using analog CMOS design techniques.

## Project Overview
The goal of this assignment is to understand the basic OTA architecture and its operation using a minimal transistor count. The design emphasizes simplicity, low power consumption, and fundamental performance analysis through simulation.

The complete design and verification were performed using **Cadence Virtuoso**, following standard analog IC design methodology.

## Assignment Objectives
- Design a 5T OTA using CMOS technology
- Analyze the operating regions of transistors
- Study gain and basic amplifier behavior
- Perform schematic-level simulations
- Interpret simulation results
